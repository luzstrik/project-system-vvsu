# Анализ предметной области

## Введение

**Тема:** информационная система спортивных организаций города.

**Название системы:** SportCity

**Планируемые интеграции:** городские порталы, государственные реестры
спортивных объектов, системы онлайн-регистрации на соревнования.

**Описание:**\
система предназначена для учета спортивных сооружений города,
спортсменов, тренеров, клубов, соревнований и организаторов. Она
обеспечивает хранение информации о спортивной инфраструктуре, участниках
соревнований и результатах их выступлений, а также позволяет формировать
аналитические выборки по различным критериям.

------------------------------------------------------------------------

## Предметная область

### Целевая аудитория

-   Администрация города
-   Руководители спортивных организаций
-   Организаторы соревнований
-   Тренеры
-   Сотрудники спортивных клубов

Система ориентирована на структурированное хранение данных о спортивной
деятельности города и формирование отчетности.

------------------------------------------------------------------------

## Хранимые сущности

### Сущности и атрибуты

| Сущность | Атрибуты |
|----------|----------|
| Спортивное сооружение | Название, адрес, тип, вместимость (для стадиона), тип покрытия (для корта), площадь, год ввода в эксплуатацию |
| Тип сооружения | Название (стадион, корт, манеж, зал) |
| Вид спорта | Название |
| Спортсмен | ФИО, дата рождения, разряд, клуб |
| Тренер | ФИО, квалификация, вид спорта |
| Клуб | Название, адрес |
| Организатор | Название, контактная информация |
| Соревнование | Название, дата начала, дата окончания, вид спорта, сооружение, организатор |
| Участие | Спортсмен, соревнование, результат, занятое место |
| Награждение | Спортсмен, соревнование, награда |

## Особенности предметной области

-   Спортивные сооружения имеют общий набор атрибутов и специфические
    характеристики в зависимости от типа.
-   Один спортсмен может заниматься несколькими видами спорта.
-   В рамках одного вида спорта спортсмен может тренироваться у
    нескольких тренеров.
-   Один спортсмен состоит только в одном клубе.
-   Соревнования проводятся по одному виду спорта.
-   Соревнование проводится на одном спортивном сооружении.
-   Организатор может проводить несколько соревнований.
-   По итогам соревнований спортсмены получают награды.

------------------------------------------------------------------------

## Проблема и текущее состояние

В большинстве городов учет спортивной деятельности ведется разрозненно:

-   информация о сооружениях хранится отдельно;
-   данные о спортсменах и клубах --- в локальных базах;
-   учет соревнований --- в отчетных документах.

Отсутствует единая система, позволяющая:

-   централизованно учитывать спортивную инфраструктуру;
-   отслеживать участие спортсменов в соревнованиях;
-   анализировать активность клубов;
-   формировать статистику по тренерам и организаторам.

## Цель и задачи проекта

### Цель

Проектирование информационной системы для централизованного учета
спортивной инфраструктуры и спортивной деятельности города.

### Основные задачи

-   Проанализировать предметную область.
-   Определить структуру данных.
-   Спроектировать модель базы данных.
-   Реализовать механизм формирования запросов.
-   Обеспечить формирование аналитической отчетности.
-   Разработать пользовательский интерфейс.

------------------------------------------------------------------------

## Основные требования

### Функциональные требования

Система должна обеспечивать возможность:

1.  Получения перечня спортивных сооружений указанного типа или
    удовлетворяющих заданным характеристикам.
2.  Получения списка спортсменов по виду спорта и/или разряду.
3.  Получения списка спортсменов конкретного тренера.
4.  Определения спортсменов, занимающихся более чем одним видом спорта.
5.  Получения списка тренеров конкретного спортсмена.
6.  Получения перечня соревнований за период или по организатору.
7.  Получения списка призеров соревнования.
8.  Получения соревнований по сооружению или виду спорта.
9.  Получения перечня клубов и количества их спортсменов, участвовавших
    в соревнованиях за период.
10. Получения списка тренеров по виду спорта.
11. Получения спортсменов, не участвовавших в соревнованиях за период.
12. Получения организаторов и количества проведенных ими соревнований.
13. Получения перечня спортивных сооружений и дат проведения
    соревнований на них.
